1.	Instalirajte PostgreSQL za vaš sustav (https://www.postgresql.org/download/)
2.	Tijekom instalacije postavite lozinku "internship" ili je promjenite sa naredbom u SQL-u kod petog koraka (ALTER USER postgres WITH PASSWORD 'internship';)
3.	Nakon instalacije otvorite pgAdmin4.
4.	Kreirajte novi database, ime baze neka bude 'ScheduleIT-DB'
5.	Desni klik na kreiranu bazu --> Query Tool --> Copy paste podatke iz file-ova "ScheduleIT-DB SQL kod" i "mock unosi"
6.	Provjerite dali su učitane tablice (ime_baze --> Schemas --> public --> Tables)
